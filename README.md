# ⚡ Josify — 補助金・助成金 一発申請サポートAI

> 事業者情報を入力するだけで、使える補助金を自動マッチングし、必要書類の下書きまで一気に完成させる AI ツール。

---

## 何ができるか

1. **マッチング** — 業種・規模・目標を入力すると、Claude AIが15種類の補助金・助成金から適合スコアを算出して提案
2. **書類準備** — 選択した補助金の必要書類を3タイプで整理:
   - `AI下書き` — 事業計画書・取組計画書など記述書類をClaude Sonnetが自動生成
   - `公式フォーム` — 官公庁の公式様式へのリンク
   - `取得ガイド` — 決算書・登記簿など第三者書類の入手方法を案内
3. **ダウンロード** — 生成した下書きをテキストファイルでダウンロード

---

## 対応している補助金・助成金（15種類）

| カテゴリ | 補助金名 |
|---|---|
| 設備投資 | ものづくり補助金 |
| DX・IT化 | IT導入補助金、中小企業DX補助金 |
| 販路開拓 | 小規模事業者持続化補助金 |
| 新規事業 | 事業再構築補助金 |
| 雇用・人材 | キャリアアップ助成金、人材開発支援助成金 |
| 賃上げ | 業務改善助成金 |
| 労務改善 | 働き方改革推進支援助成金 |
| 育児・介護 | 両立支援等助成金 |
| 省エネ | 省エネルギー投資促進支援事業費補助金 |
| 事業承継 | 事業承継・引継ぎ補助金 |
| 研究開発 | スタートアップ・研究開発補助金（SBIR） |
| 地域経済 | 地域未来投資促進補助金 |
| 脱炭素 | グリーンイノベーション基金事業 |

## セットアップ

### 1. 必要なもの
- Node.js 18以上
- Anthropic API キー（[console.anthropic.com](https://console.anthropic.com) で取得）

### 2. インストール

```bash
cd josify
npm install
```

### 3. 環境変数の設定

```bash
cp .env.example .env
```

`.env` を開いて API キーを入力:

```
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxx
```

### 4. 起動

```bash
npm start
```

ブラウザで `http://localhost:3000` を開く。

---

## 使い方

1. **Step 1**: 業種・従業員数・所在地・年商・目標を入力して「補助金を検索」
2. **Step 2**: マッチした補助金が一覧表示される → 申請したいものを選択
3. **Step 3**: AIが各書類の下書きを自動生成（2〜3分かかる場合あります）
4. **Step 4**: 下書きをダウンロードして専門家（社労士・税理士・中小企業診断士）に確認依頼

---

## ⚠️ 重要な免責事項

- このツールが生成する書類はAIによる**ドラフト（下書き）**です
- 申請前に必ず**税理士・社労士・中小企業診断士等の専門家**に内容を確認してください
- 補助金の公募期間・要件・金額は**随時変更**されます。必ず**公式サイトで最新情報**を確認してください
- `【要確認】` `【要入力】` の箇所は実際の数値・情報に置き換えてください
- 本ツールの使用により生じた損害について、開発者は一切責任を負いません
